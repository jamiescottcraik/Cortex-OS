# Providers & Setup

## MLX
- Start MLX service and set `MLX_SERVICE_URL`.
- Optionally configure `MLX_CACHE_DIR` for model storage.

## Ollama
- Install Ollama and set `OLLAMA_URL`.

## LangChain / CrewAI
- Ensure Python dependencies are installed via `requirements.txt`.
