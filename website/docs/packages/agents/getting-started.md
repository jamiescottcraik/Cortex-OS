# Getting Started

## Prerequisites
- Node.js 18+
- pnpm 8+

## Installation
```bash
pnpm install
pnpm build --filter @cortex-os/agents
```

## First Launch
Import the package and register an agent:
```typescript
import { Agent } from '@cortex-os/agents';
// see README usage for a full example
```
