# Getting Started

## Prerequisites
- Node.js ≥ 20
- PNPM ≥ 10

## Installation
```bash
pnpm install
```

## First Launch
Start the development server on port 3333:
```bash
pnpm --filter @cortex-os/gateway dev
```
Access `http://localhost:3333/health` to verify the server is running.
