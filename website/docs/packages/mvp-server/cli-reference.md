# CLI Reference

```
mvp-server [command] [options]
```

## Commands
| Command | Description |
| --------- | ------------- |
| `start` | Launch server in production mode |
| `dev` | Start with hot-reload |
| `build` | Bundle plugins for deployment |
| `health` | Run readiness check |

## Global Options
| Option | Description |
| -------- | ------------- |
| `-c, --config &lt;path&gt;` | Path to config file |
| `-p, --port &lt;number&gt;` | Override port |
| `-l, --log-level &lt;level&gt;` | Set log level |
