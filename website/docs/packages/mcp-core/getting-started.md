## First Tool Call

```ts

const client &#61; await createEnhancedClient({
  name: "example",
  transport: "streamableHttp",
  endpoint: "http://localhost:3000/tool"
});

const result &#61; await client.callTool({ name: "ping" });
await client.close();
```
