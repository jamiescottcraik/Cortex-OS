# Contributor Setup

1. Install dependencies:
   ```bash
   pnpm install
```
2. Build the package:
   ```bash
   pnpm --filter @cortex-os/evals build
```
3. Run tests:
   ```bash
   pnpm --filter @cortex-os/evals test
```

Follow the repository's [CONTRIBUTING](/docs/references/contributing guidelines for commit style and pull requests.
