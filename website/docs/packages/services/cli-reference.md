# CLI Reference

## Model Gateway
- `pnpm dev` - start development server.
- `pnpm build` - compile TypeScript to `dist`.
- `pnpm start` - run compiled server.
- `pnpm test` - execute unit tests.

## Orchestration
- `pnpm build` - compile TypeScript library.
- `pnpm test` - run unit tests.
