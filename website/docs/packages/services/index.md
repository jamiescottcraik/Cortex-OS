# Cortex Service Packages Documentation

Central entry point for Cortex-OS service packages. These services expose runtime capabilities such as model access and workflow orchestration.

## Features

- **Model Gateway** - Fastify API for model inference with schema validation and streaming responses.
- **Orchestration Engine** - DAG-based workflow executor with hooks, compensation, and human-in-the-loop support.

### Planned

- **Telemetry Service** - unified metrics and tracing collection.
