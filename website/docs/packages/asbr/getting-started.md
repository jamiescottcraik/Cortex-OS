# Getting Started

## Prerequisites
- Node.js \u2265 18
- pnpm package manager

## Installation
```bash
pnpm install @cortex-os/asbr
```

## First Launch
Start the runtime with the built-in CLI:
```bash
npx cortex-asbr
```
The command prints the local URL and an access token.
