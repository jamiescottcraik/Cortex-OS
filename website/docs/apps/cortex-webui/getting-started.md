# Getting Started

Follow these steps to run Cortex WebUI locally.

## Prerequisites

- Node.js >= 16
- pnpm >= 7
- Git

## Installation

```bash
pnpm install
```

## First Launch

Start both backend and frontend in development mode:

```bash
pnpm dev
```

Frontend runs on `http://localhost:3000` and backend on `http://localhost:3001`.
