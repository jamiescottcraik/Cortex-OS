# Contributor Setup

1. Install dependencies:
   ```bash
   pnpm install
   ```
2. Build the package:
   ```bash
   pnpm --filter @cortex-os/observability build
   ```
3. Run lint and tests:
   ```bash
   pnpm --filter @cortex-os/observability lint
   pnpm --filter @cortex-os/observability test
   ```
