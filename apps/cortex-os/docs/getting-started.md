# Getting Started

## Prerequisites
- Node.js ≥ 20 and pnpm ≥ 9
- Optional: Python ≥ 3.11 with `uv` for MLX backends

## Installation
```bash
pnpm install
pnpm build
```

## First Launch
```bash
pnpm dev
```

Set required environment variables before launching (see [Configuration](./configuration.md)).
